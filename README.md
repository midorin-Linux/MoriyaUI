# MoriyaUI

MoriyaUIは、モダンで直感的なAIチャットインターフェースを提供するTauri2.0アプリケーションです。
Shadcn UIとTailwind CSSを採用し、美しく機能的なメッセージ入力コンポーネントを中心に構築されています。

## 技術スタック

- **Frontend Framework**: Tauri2.0 + React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Icons**: Lucide React

## セットアップ

プロジェクトをローカル環境で実行するには、以下の手順に従ってください。

### 1. リポジトリのクローン

```bash
git clone https://github.com/midorin-Linux/MoriyaUI
cd MoriyaUI
```

### 2. 依存関係のインストール

```bash
bun install
```

### 3. 開発サーバーの起動

```bash
bun run tauri dev
```
